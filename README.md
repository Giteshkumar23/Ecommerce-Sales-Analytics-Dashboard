# 📊 E-Commerce Sales Analytics Dashboard

A business intelligence dashboard for analyzing e-commerce sales using Python and Streamlit.

## ✨ Features

- ETL Pipeline
- Sales Dashboard
- Revenue Analysis
- Product Insights
- Customer Analytics
- Interactive Charts

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- PostgreSQL
- Plotly

## 🚀 Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
